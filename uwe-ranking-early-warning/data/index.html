UK University Ranking Early Warning System

A machine learning dashboard predicting institutional rank, explaining the drivers behind ranking changes, and flagging universities at risk of decline before published rankings reflect it. Case study: UWE Bristol, 2022–2027.

130 UK institutions 6 years of CUG data (2022–2027) 649 observations Random Forest & XGBoost SHAP explainability

Project overview

UWE Bristol's Guardian University Guide ranking climbed from 64th to 24th place between 2011 and 2023, then fell back to 64th by 2025 — a loss of 40 places in two years. This project investigates whether such a decline could have been predicted in advance using institutional performance data.

UWE Bristol ranking trajectory (CUG, 2022–2027)

Key finding: the sharpest single-year decline occurred between 2024 and 2025, when student satisfaction dropped 0.86 points (on CUG's 1–4 scale) and the staff-student ratio worsened from 15.7 to 16.8. Using only 2024 data, the volatility classifier correctly predicted that UWE Bristol would be classified "at risk" in 2025 — before this decline appeared in any published league table.

Regression model comparison

Three models were trained to predict institutional rank, using a temporal split: 2022–2024 for training (271 rows), 2025–2027 for testing (378 rows). This reflects realistic forecasting — the model only ever sees the past when predicting the future.

Why Ridge shows 0.00 MAE: this is overfitting, not a genuinely superior model. Ridge exploits the near-perfect linear correlation between a university's rank last year and this year, which doesn't reflect real forecasting ability. Random Forest and XGBoost, which model this relationship as informative but not deterministic, are the project's primary models.

Headline result: using only data available at the end of 2024 — before UWE Bristol's 2025 ranking decline appeared in any published league table — the classifier correctly predicted UWE would be "at risk" the following year. This result held consistently across the baseline, SMOTE, and threshold-tuned classifier variants.

The 2026 prediction was incorrect, which is discussed as a limitation: the overall classifier accuracy across all 130 institutions (64%, 37% at-risk recall after tuning) remains constrained by class imbalance — only 74 of 510 next-year labels are "at risk". This is identified as the primary direction for future work, requiring a longer time series (10+ years) to provide enough examples of decline events.

Data source

Complete University Guide (CUG) league tables, 2022–2027, covering approximately 130 UK institutions per year. Selected over the Guardian University Guide because CUG provides consistent multi-year CSV downloads, enabling proper temporal train/test validation.

Pipeline

Raw data collected910 university-year observations, 13 columns
After cleaning910 rows, 14 standardised columns (column & naming inconsistencies resolved)
After feature engineering649 rows, 24 columns (lag, change, rolling-average features added)
Train / test split2022–2024 (271 rows) / 2025–2027 (378 rows), split by year
